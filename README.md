# CRM Assessment

A small CRM lead management system with a FastAPI backend and React frontend. The app supports demo login, lead CRUD, notes, and dashboard metrics.

## Source Code

The source code is in the `backend/` and `frontend/` folders.

## Quick Summary

- Backend: Python, FastAPI, sqlite3
- Frontend: React, axios, react-router-dom
- Database: SQLite (`backend/crm.db`)
- Demo auth: hardcoded token-based login for assessment purposes
